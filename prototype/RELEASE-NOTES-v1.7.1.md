# Insight Asset OS · V1.7.1

V1.7.1 是一个**小版本**——重构写作入口，把"开始写作"和"主题文章"两个相似入口合并成一个。

主要交付：
- **/write 合并入口** —— 写全文（V1.7.1 可用）+ 列提纲（V1.9 上线）
- **Sidebar 单入口** —— "写文章" 一个入口替代两个
- **/topic-articles/new 重定向** —— 老路由自动跳到 /write
- **/topic-articles/new/page.tsx 简化** —— 从 502 行到 9 行（redirect only）

## 🎉 这版推荐升

V1.7 用户**强烈推荐升**——Sidebar 更清爽，单入口避免"哪个是写文章的"困惑。
V1.6 用户**可选升**——核心功能没变，只是入口简化。

## 🆕 /write 合并入口

**痛点**：V1.7 同时有「开始写作」和「主题文章」两个入口——都是"选主题 + 卡片 → 生成"，操作流程几乎一样。Vincent 抓"两个操作步骤太相似了"——**合并成一个**。

### 新版设计

```
/write
   ↓
两个 Tab:
├── 写全文（V1.7.1 可用）── AI 直接写完整文章
└── 列提纲（V1.9 上线）── AI 给结构，你写正文
```

**共享部分**：
- 选主题
- 勾选卡片
- 系统自动注入 Kernel + Preset

**写全文专属**（V1.7.1）：
- 篇幅 select（settings 配默认）
- 篇数 select（1-5 篇）
- 调 /api/topic-articles/generate

**列提纲专属**（V1.9）：
- 模板 select（公众号/演讲/读书 + 小红书/知乎/客户方案）
- 选核心判断（来自 kernel.coreBeliefs）
- 调 /api/writing/scaffold

### UI 细节

| 元素 | 设计 |
|------|------|
| **顶部系统注入条** | 显示「📐 Kernel · ✍️ vincent-standard」+ 跳 settings 链接 |
| **Tab 视觉差异** | 「写全文」实色高亮（紫蓝渐变）；「列提纲」灰虚线 + "V1.9 上线" 角标 |
| **副标题** | "选主题 → 勾卡片 → AI 帮你生成文章系列" |

### 兼容性

- `/topic-articles/new` 自动重定向到 `/write`（307 永久重定向）
- `/writing/new` 保留（V1.7.1 不动，因为列提纲 tab 是 V1.9 上线）
