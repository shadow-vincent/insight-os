# Vincent 的核心内核 · 最后更新 2026-06-21

## 底层信念
- 管理的本质是激发人的善意和潜能，而不是控制
- 工作的意义在于创造，不是消耗
- 判断力比知识稀缺，AI 时代尤其如此
- 管理咨询的价值不是给答案，是帮客户看见自己的问题

## 反常识判断
- AI 不会让所有人变强，只让强者更强
- 组织变革的瓶颈从来不是技术，是组织吸收力
- Prompt engineering 不是 AI 时代的关键能力，是判断力训练

## 擅长问题域
- 组织变革（特别是大企业从 0 到 1 推动新业务）
- 战略定位（特别是 AI 时代差异化战略）

## 想挑战的行业常识
- "AI 让所有人变强" — 这是 AI 厂商营销叙事，不是事实
- "赋能" / "抓手" / "闭环" — 这些词是空话，不是洞察

🎯 关键设计点

✅ 借鉴 Claude.md

  • 极简 — 一个文件 + 一份结构化数据
  • 自动加载 — 每次 LLM 调用注入
  • 用户掌控 — UI + 可手动编辑 .md
  • git 版本控制 — 可分享可 fork

⭐ 比 Claude.md 强

  • 结构化 4 类 — 比纯 markdown 精确
  • 每条带 metadata — 量化引用次数
  • weekly reflection — 自动建议
  • per-user 隔离 — 不共享
  • 跨 10 个场景 — 不只是 chat

🔐 数据隔离

  • 每个用户独立 core_beliefs 表
  • 登录后只加载自己的内核
  • 不共享、不上传云端（自用为主）

🔄 闭环

  • LLM 输出 → feedback 表
  • weekly reflection → 自动建议
  • 人工采纳 → 内核更新
  • 下次 LLM 调用 → 新立场
